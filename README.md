diary-app

Laravelで作成するシンプルな日記アプリです。
PHP + MariaDB を使用したWebアプリケーション開発の学習目的で作成しています。
Webアプリケーションの基本的な構成や開発フローの理解を目的に開発しています。

⸻

■ 使用技術

・PHP

・Laravel

・MariaDB

・Nginx

・AWS（EC2 / RDS）※予定

⸻

■ 機能

・日記一覧表示

・日記投稿

・日記編集

・日記削除

シンプルなCRUD機能を中心に実装予定です。
